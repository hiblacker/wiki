### 网络

-   HTTP: 强缓存/协商缓存
-   HTTP: 状态码
-   HTTP 与 HTTPS 区别
    -   HTTP 是明文传输，HTTPS 的数据传输是经过 SSL 加密的，更加安全。
    -   HTTP 使用 TCP 三次握手交换三个包建立连接，HTTPS 除了三次握手还需要 SSL 握手，需要交换 9 个包，因此速度慢于 HTTP，更耗费服务器资源。
    -   HTTP 默认 80 端口，HTTPS 默认 443 端口
-   HTTPS 加密原理
-   TCP: 连接建立流程：三次握手、四次挥手
